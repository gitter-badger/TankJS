    Entity

    An Entity represents a single game object or container. It contains Components and children Entities, and can also dispatch events. An Entity’s functionality is defined by the set of Components it has, and can be used to represent the player, an object spawner, a trigger zone, or really just about anything. Since Entities can contain child Entities, they can be used to create hierachical structures, or to separate portions of your game that have separate behavior, since different Entities can send separate events and be paused individually.
